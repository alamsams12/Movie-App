# 🎬 Movie App

A React movie application built with **Vite** using the **TMDB API**.
The app allows users to browse movies, check release dates, and save their favorite movies.

---

## 🚀 Features

* Browse popular movies
* View movie release dates
* Add movies to favorites
* Remove movies from favorites
* Clean and responsive UI

---

## 🛠 Tech Stack

* React.js
* Vite
* JavaScript
* TMDB API
* CSS

---

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/alamsams12/Movie-App.git
```

2. Navigate to the project folder

```
cd Movie-App
```

3. Install dependencies

```
npm install
```

4. Run the project

```
npm run dev
```

---

## 🌐 API Used

This project uses the **TMDB (The Movie Database) API** to fetch movie data.

https://www.themoviedb.org/
